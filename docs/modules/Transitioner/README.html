Transitioner

シーン遷移アニメーションを行うためのモジュールです。
実行すると「シーンの遷移」「遷移に伴うローディング画面の表示」を行います。
ローディング画面のテクスチャリソースは任意のものが利用できます。

内部実装的にはSceneStateManager と FadingTextureResource を利用した VaNilla.Objectです。

APIリファレンス

詳細なAPIドキュメントはVaNilla Transitioner APIリファレンスをご覧ください。

使い方

SceneTransitioner

下記「FadeTransitioner」「EasyTransitioner」を簡単に使うためのユーティリティクラスです。
大体のケースではこのクラスの SceneTransitioner.Transit を利用すれば事足りるはずです。

FadeTransitioner

指定したローディング画面用のテクスチャリソースを挟んでフェードイン・フェードアウトを行います。
ローディング画面の表示中に、シーン切り替えを実施します。
本VaNilla.Objectはシーン切り替えおよびフェードアウト完了後、自動的に破棄されます。

EasyTransitioner

未実装です。
