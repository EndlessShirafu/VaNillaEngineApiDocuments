VaNillaTestProject

VaNilla全体の開発テストを行うためのリポジトリです。

Common

プラットフォーム共通で利用するソースコード類が配置されています。

DxLib

DxLib用のテストプロジェクトが配置してあります。
実行方法は 個別のREADMEテキスト をご参照ください。

Unity

Unity用のテストプロジェクトが配置してあります。

[Scenes] -> [Test] がテスト用のシーンとなっており、
ロード後にPlayすることでテストが順次実行されてログが出力されます。
Unityのデバッグコンソールウインドウ上にsome Test(s) are failed!というエラーログが出力されていた場合、テストが失敗していることを意味します。
逆にAll Test(s) are passed!と出ていたらテストが成功しています。

現状で実行するテストは以下です。

PseudoTest

  • 何もしないテストです。
  • テストのテストといった用途です。

SceneStateTest

  • SceneStateManagerによるシーン遷移が意図した挙動になっているかどうかをチェックするテストです。

ObjectExecutionPriorityTest

  • ObjectExecutionPriority, ExpresserComponentExecutionOrder が正常動作しているかどうかをチェックするテストです。
