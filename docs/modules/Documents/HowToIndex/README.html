基本的なゲーム開発

⚙️ ゲームの起動とメインループの制御をしたい

→ SupremeArbiter - メインループ制御

🎮 ゲーム起動時の初期化処理を作りたい

→ BootLoader - ゲームプログラムの起動

🎯 ゲームオブジェクトを作りたい

→ Object - 基本的なオブジェクトの作成

🏭 大量のオブジェクトを効率的に管理したい

→ ObjectFactory - ファクトリーパターンでのオブジェクト生成 → ObjectPool - オブジェクトプールの実装

🎬 シーン遷移を実装したい

→ SceneStateManager - シーン管理システム → Transitioner - シーン遷移エフェクト

グラフィックス・描画

🖼️ 画像を表示したい

→ TextureResourceManager - テクスチャの読み込みと管理 → Renderer - 画像を使った描画

📝 テキストを表示したい

→ FontResourceManager - フォント管理とテキスト描画

🎨 オフスクリーンレンダリングを使いたい

→ OffScreenRenderer - レンダーターゲットの管理 → OffScreenResourceManager - オフスクリーンリソース管理

🌈 ブレンドモードを変更したい（アルファ合成・加算合成・乗算合成・減算合成）

→ ShaderFactory - サポートされるブレンドモード

🔲 9スライス画像を使いたい

→ NineSliceRenderer - 9スライススプライトの描画

📐 図形を描画したい

→ Renderer - 基本図形の描画

入力処理

⌨️ キーボードの入力を取得したい

→ KeyboardInput - キーボード入力の処理

🖱️ マウスの入力を取得したい

→ MouseInput - マウス入力の処理 → MouseReact - マウス追従機能

🎮 ゲームパッドの入力を取得したい

→ GamePadInput - ゲームパッド入力の処理

⚙️ キーコンフィグを実装したい

→ KeyConfig - キー設定の管理

⚙️ キーコンフィグした内容で取得するキー・ボタンを呼び分けたい

→ PlayerInput - StaticInputInfo

🖱️ クリック可能な領域（ボタンなど）を作りたい

→ ClickableRegion - クリック領域の実装

音声・サウンド

🔊 効果音を再生したい

→ SoundManager - サウンドの再生と管理

🎵 BGMをループ再生したい

→ SoundManager - BGMのループ再生

物理・衝突判定

💥 当たり判定を実装したい

→ Collision - 衝突判定システム

🏃 オブジェクトを移動させたい

→ Movement - 各種移動パターン → Transform - 座標・回転・スケール管理

🌀 物理演算を使いたい

→ Physics - 物理エンジン統合

UI・インターフェース

📝 テキストを表示したい

→ FontResourceManager - フォント管理とテキスト描画

🌐 多言語対応したい

→ Localization - ローカライゼーション

📊 FPSを表示したい

→ FpsCalculator - フレームレート計算

🎯 画面解像度を管理したい

→ ScreenResolutionManager - 解像度管理

🪟 ウィンドウサイズを変更したい

→ WindowSizeManager - ウィンドウサイズ管理

データ管理

💾 セーブデータを暗号化したい

→ Encrypter - データの暗号化

📁 アセットを管理したい

→ AssetManipulator - アセット操作

📊 読み取り専用データベースを使いたい

→ ReadOnlyTextDataBaseManager - テキストDB管理

🎲 乱数を生成したい

→ RandomNumberGenerator - 乱数生成

📅 デイリーシステムを実装したい

→ DailySystem - デイリーミッション・ログインボーナス

ネットワーク・通信

🏆 Steamworksと連携したい

→ SteamworksManager - Steam API統合

📤 パッチを配信したい

→ PatchUtil - パッチシステム

デバッグ・開発支援

📝 ログを出力したい

→ DebugLogger - ログ出力システム

🔍 デバッグ情報を表示したい

→ DebugUtil - デバッグユーティリティ

⏱️ 処理時間を計測したい

→ TimeManager - 時間管理とタイマー

📹 ゲーム画面を録画したい

→ MovieDecoder - 動画デコード機能

ゲームロジック

→ ConditionalProcessor - 条件処理システム

📈 イージング関数を使いたい

→ Easing - イージング関数集

🔄 非同期タスクを使いたい

→ Task - 非同期タスクの実行
