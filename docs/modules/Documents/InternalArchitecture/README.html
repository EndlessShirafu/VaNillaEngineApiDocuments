SharedProjectSnippets構造についての重要な注意

VaNillaのSharedProjectSnippetsは、実装が依存するライブラリごとに厳密に分離されています：

SharedProjectSnippets/
├── Common/     # プラットフォーム非依存のインターフェース層
├── DxLib/      # DxLib（DirectX Library）に依存する実装
├── Box2D/      # Box2D物理エンジンに依存する実装
└── [他のライブラリ]/  # その他の外部ライブラリ依存

誤解を避けるために：

  • DxLib/ = DxLibというライブラリに依存する実装（DxLibフォルダという意味ではない）
  • Box2D/ = Box2Dというライブラリに依存する実装
  • 新しい外部ライブラリを追加する際は、そのライブラリ名のディレクトリを作成

注意事項

  • これらのドキュメントは内部仕様を扱うため、将来のバージョンで変更される可能性があります
  • 通常のゲーム開発では、これらの内部仕様を意識する必要はありません
  • 内部実装に依存したコードは、エンジンのアップデート時に動作しなくなる可能性があります
