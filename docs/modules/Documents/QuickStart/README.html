VaNillaエンジンとは

VaNillaは、DxLibやUnity上で動作するメタ・ゲームエンジンです。
2Dアクションゲームやシューティングゲームの開発に特化しており、以下の特徴があります：

  • 🎯 統一API - DxLibとUnityで同じコードが動作
  • ⚡ 高性能 - 数千個のオブジェクトを軽量に処理
  • 🧩 ExpresserComponent設計 - 「何を表現するか」でゲームオブジェクトを構成
  • ⚙️ ゼロコスト事前定義 - YAML設定・属性宣言からC#コードを事前生成

ExpresserComponent-Oriented Programming（表象要素指向プログラミング）

VaNillaの核となる設計思想です。ゲームオブジェクトを「何を持っているか」ではなく「何を表現するか」で構成します。

例えば、プレイヤーキャラクターは：

  • 座標を持つ（HasPosition）
  • 画像を持つ（HasTexture）
  • 当たり判定がある（HasCollision）
  • 移動できる（PlayerMovementComponent）

これらの「表象要素」を組み合わせてゲームオブジェクトを作ります。

git clone https://github.com/EndlessShirafu/VaNilla.git
cd VaNilla

プロジェクトのセットアップ

1. 設定ファイルの準備

まず、ゲーム用の設定ファイル（YAML）を作成します。

VaNillaTestProject/DxLib/VaNillaSettings.yamlを開いて、自分のプロジェクト用に編集します：

# テクスチャリソースの定義
textures:
  - name: Player
    path: "data/player.png"
  - name: Enemy
    path: "data/enemy.png"
  - name: Bullet
    path: "data/bullet.png"

# シーン定義
scenes:
  - name: Title
  - name: Game
  - name: GameOver

# 実行順序の定義
execution_order:
  - name: Input
    priority: 100
  - name: Logic
    priority: 200
  - name: Rendering
    priority: 300

2. コード生成（重要！）

VaNillaエンジンはYAMLからC#コードを生成します。この手順を飛ばすとビルドエラーになります。

# プロジェクトルートで実行
python VaNillaEngineManager/CodeGenerator.py -s VaNillaTestProject/DxLib/VaNillaSettings.yaml

これにより以下が生成されます：

  • TextureResourceType 列挙型
  • SceneStateType 列挙型
  • ExpresserComponentExecutionOrder 列挙型
  • その他の設定クラス

3. プロジェクトのビルド

# DxLib版をビルド
cd VaNillaTestProject/DxLib
dotnet build

最初のゲームオブジェクト

プレイヤーキャラクターを作る

Player.csを作成：

using System;
using VaNilla;
using VaNilla.Transform;
using VaNilla.Texture;

namespace MyGame
{
    /// <summary>
    /// プレイヤーキャラクター
    /// </summary>
    [AutoGenerateConstructor]
    [DefaultExpresserComponent(typeof(HasPosition), auto_member_generate: true)]
    [DefaultExpresserComponent(typeof(HasTexture), auto_member_generate: true)]
    public partial class Player : Object
    {
        public override bool SetUp()
        {
            // 初期位置を画面中央に設定
            hasPosition.Position = new Point2(400, 300);
            
            // テクスチャを設定（YAMLで定義したものを使用）
            hasTexture.Texture = TextureResourceManager.GetTextureResource(TextureResourceType.Player);
            
            return true;
        }
        
        public override bool Execute(in Time.Time time)
        {
            // 毎フレーム実行される処理
            return true;
        }
    }
}

属性の説明

  • [AutoGenerateConstructor] - コンストラクタを自動生成
  • [DefaultExpresserComponent] - このオブジェクトが持つExpresserComponentを指定
  • auto_member_generate: true - メンバー変数（hasPosition, hasTexture）を自動生成

プレイヤーを画面に表示

Main.csに追加：

[GameBootLoader]
public static bool InitializeGame()
{
    // プレイヤーを生成
    ObjectActivator.Activate<Player>();
    return true;
}

ExpresserComponentで機能を追加

移動機能を実装

PlayerMovementComponent.csを作成：

using System;
using VaNilla;
using VaNilla.Input;
using VaNilla.Transform;

namespace MyGame
{
    /// <summary>
    /// プレイヤーの移動処理
    /// </summary>
    [RequiredExpresserComponent(typeof(HasPosition), auto_member_generate: true)]
    [ExpresserComponentExecutionOrderAttribute(ExpresserComponentExecutionOrder.Logic)]
    public partial class PlayerMovementComponent : IExpresserComponent
    {
        private const float MoveSpeed = 5.0f;
        
        public bool Execute(Object obj, in Time.Time time)
        {
            // キーボード入力で移動
            if (KeyboardInput.IsPushed(KeyCode.Left))
                hasPosition.Position.X -= MoveSpeed;
            if (KeyboardInput.IsPushed(KeyCode.Right))
                hasPosition.Position.X += MoveSpeed;
            if (KeyboardInput.IsPushed(KeyCode.Up))
                hasPosition.Position.Y -= MoveSpeed;
            if (KeyboardInput.IsPushed(KeyCode.Down))
                hasPosition.Position.Y += MoveSpeed;
            
            // 画面外に出ないように制限
            hasPosition.Position.X = Math.Clamp(hasPosition.Position.X, 0, 800);
            hasPosition.Position.Y = Math.Clamp(hasPosition.Position.Y, 0, 600);
            
            return true;
        }
        
        public bool SetUp(Object vanillaObject) => true;
        public void TearDown(Object vanillaObject) { }
    }
}

入力処理を実装

弾の発射機能

Bullet.csを作成：

using System;
using VaNilla;
using VaNilla.Transform;
using VaNilla.Texture;
using VaNilla.Movement;

namespace MyGame
{
    [AutoGenerateConstructor]
    [DefaultExpresserComponent(typeof(HasPosition), auto_member_generate: true)]
    [DefaultExpresserComponent(typeof(HasTexture), auto_member_generate: true)]
    [DefaultExpresserComponent(typeof(LinearUniformMotion), auto_member_generate: true)]
    public partial class Bullet : Object
    {
        public void Initialize(float x, float y, float speedY)
        {
            hasPosition.Position = new Point2(x, y);
            hasTexture.Texture = TextureResourceManager.GetTextureResource(TextureResourceType.Bullet);
            
            // 上方向に移動
            linearUniformMotion.Velocity = new Velocity2(0, speedY);
        }
        
        public override bool Execute(in Time.Time time)
        {
            // 画面外に出たら自分を削除
            if (hasPosition.Position.Y < -50)
            {
                ObjectActivator.Deactivate(this);
            }
            
            return true;
        }
    }
}

プレイヤーに発射機能を追加

PlayerShootingComponent.csを作成：

using System;
using VaNilla;
using VaNilla.Input;
using VaNilla.Transform;

namespace MyGame
{
    [RequiredExpresserComponent(typeof(HasPosition), auto_member_generate: true)]
    [ExpresserComponentExecutionOrderAttribute(ExpresserComponentExecutionOrder.Logic)]
    public partial class PlayerShootingComponent : IExpresserComponent
    {
        private float shootCooldown = 0;
        private const float ShootInterval = 0.1f; // 0.1秒ごとに発射可能
        
        public bool Execute(Object obj, in Time.Time time)
        {
            shootCooldown -= time.DeltaTime;
            
            // スペースキーで弾を発射
            if (KeyboardInput.IsPushed(KeyCode.Space) && shootCooldown <= 0)
            {
                var bullet = ObjectActivator.Activate<Bullet>();
                bullet.Initialize(hasPosition.Position.X, hasPosition.Position.Y - 20, -10);
                
                shootCooldown = ShootInterval;
            }
            
            return true;
        }
        
        public bool SetUp(Object vanillaObject) => true;
        public void TearDown(Object vanillaObject) { }
    }
}

描画処理

VaNillaでは、HasTextureとBasicRendererを組み合わせることで自動的に描画されます。

描画の仕組み

// BasicRendererの追加（自動的にHasTextureを参照して描画）
[DefaultExpresserComponent(typeof(BasicRenderer))]
public partial class GameObject : Object
{
    // BasicRendererがHasTextureのテクスチャを
    // HasPositionの座標に描画してくれる
}

カスタム描画処理

特殊な描画が必要な場合は、独自のRendererComponentを作成：

[RequiredExpresserComponent(typeof(HasPosition), auto_member_generate: true)]
[RequiredExpresserComponent(typeof(HasTexture), auto_member_generate: true)]
[ExpresserComponentExecutionOrderAttribute(ExpresserComponentExecutionOrder.Rendering)]
public partial class CustomRenderer : IExpresserComponent
{
    public bool Execute(Object obj, in Time.Time time)
    {
        // 点滅効果
        if ((int)(time.TotalTime * 10) % 2 == 0)
        {
            Renderer.Utils.DrawTexture(
                hasTexture.Texture,
                hasPosition.Position,
                1.0f, // スケール
                0.0f, // 回転
                255   // アルファ値
            );
        }
        
        return true;
    }
}

当たり判定

敵キャラクターの作成

Enemy.csを作成：

using System;
using VaNilla;
using VaNilla.Transform;
using VaNilla.Texture;
using VaNilla.Collision;

namespace MyGame
{
    [AutoGenerateConstructor]
    [DefaultExpresserComponent(typeof(HasPosition), auto_member_generate: true)]
    [DefaultExpresserComponent(typeof(HasTexture), auto_member_generate: true)]
    [DefaultExpresserComponent(typeof(HasCollision), auto_member_generate: true)]
    public partial class Enemy : Object, ICollisionCallback
    {
        public override bool SetUp()
        {
            hasPosition.Position = new Point2(400, 100);
            hasTexture.Texture = TextureResourceManager.GetTextureResource(TextureResourceType.Enemy);
            
            // 当たり判定の設定
            hasCollision.Shape = new CircleCollisionShape(16); // 半径16の円
            hasCollision.CollisionType = CollisionType.Enemy;
            
            return true;
        }
        
        public void OnCollisionEnter(CollisionResult result)
        {
            // 弾と衝突したら
            if (result.OpponentObject is Bullet)
            {
                // 自分と弾を削除
                ObjectActivator.Deactivate(this);
                ObjectActivator.Deactivate(result.OpponentObject);
            }
        }
    }
}

シーン管理

タイトルシーンの作成

TitleScene.csを作成：

using System;
using VaNilla;
using VaNilla.SceneState;
using VaNilla.Input;
using VaNilla.Renderer;

namespace MyGame
{
    [SceneStateProcess(SceneStateType.Title)]
    public class TitleScene : SceneStateProcessInterface
    {
        public void Process()
        {
            // タイトル画面の描画
            Utils.DrawString("MY FIRST GAME", 300, 200, Color.White);
            Utils.DrawString("Press SPACE to Start", 280, 300, Color.Gray);
            
            // スペースキーでゲーム開始
            if (KeyboardInput.IsPushedMoment(KeyCode.Space))
            {
                SceneStateManager.ChangeScene(SceneStateType.Game);
            }
        }
    }
}

ゲームシーンの作成

GameScene.csを作成：

using System;
using VaNilla;
using VaNilla.SceneState;

namespace MyGame
{
    [SceneStateProcess(SceneStateType.Game)]
    public class GameScene : SceneStateProcessInterface
    {
        private bool initialized = false;
        
        public void Process()
        {
            if (!initialized)
            {
                // ゲーム開始時の初期化
                ObjectActivator.Activate<Player>();
                ObjectActivator.Activate<Enemy>();
                initialized = true;
            }
            
            // ゲームオーバー条件などをここに実装
        }
    }
}

初期シーンの設定

Main.csに追加：

[GameBootLoader]
public static bool InitializeGame()
{
    // 最初はタイトルシーンから開始
    SceneStateManager.ChangeScene(SceneStateType.Title);
    return true;
}

トラブルシューティング

よくあるエラー

Q: ObjectTagEnumやTextureResourceTypeが見つからない
A: コード生成を実行してください：

python VaNillaEngineManager/CodeGenerator.py -s VaNillaSettings.yaml

Q: partialクラスにしろというエラーが出る
A: [AutoGenerateConstructor]を使うクラスはpartial宣言が必要です：

public partial class Player : Object  // partialを追加

Q: ExpresserComponentが実行されない
A: 実行順序属性を確認してください：

[ExpresserComponentExecutionOrderAttribute(ExpresserComponentExecutionOrder.Logic)]
