PerformanceProfilerTest

PerformanceProfilerモジュールの単体テストプロジェクトです。

テスト実行方法

# プロジェクトディレクトリに移動
cd VaNilla/PerformanceProfiler/Test/PerformanceProfilerTest

# テスト実行
dotnet test

# 詳細な出力付きでテスト実行
dotnet test --logger "console;verbosity=detailed"

テストカテゴリ

PerformanceProfilerTests

  • 基本的なAPI機能のテスト
  • プロファイリングの開始/停止
  • 各種メトリクス取得の動作確認

MemoryProfilingTests

  • メモリ使用量計測機能のテスト
  • GCカウント追跡の検証
  • メモリ統計の整合性確認

ProfilingHistoryTests

  • 履歴管理機能のテスト
  • サンプリング設定の動作確認
  • エッジケースの処理確認

注意事項

  • 一部のテストはシステムリソースに依存するため、環境によって結果が異なる可能性があります
  • GC関連のテストは.NETランタイムの動作に依存します
  • メモリ使用量のテストは他のプロセスの影響を受ける可能性があります
