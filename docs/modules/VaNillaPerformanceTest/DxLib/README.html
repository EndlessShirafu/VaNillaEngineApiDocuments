VaNilla Test Project for DxLib

これはなに

DxLib実装向けのVaNillaエンジンのテストを行うためのプロジェクトです。

ディレクトリ構成

  • code: DxLib実装向けのVaNillaエンジンのテスト実施に必要なソースコードが入っています。
  • DxLibDotNet: 依存ライブラリである、DxLibを配置したディレクトリです。
  • project: ビルドプロジェクトを配置したディレクトリです。
    • vs2022: Visual Studio 2022向けのプロジェクトファイルを配置したディレクトリです。

実行環境

  • Microsoft Visual Studio 2022 Version 16.11.9
  • .NET 9.0以上
    • コマンドプロンプトでdotnetコマンドが利用できる状態

実行環境のインストール方法

本READMEと同階層に存在するinstall_and_run_test.batを実行します。
自動的に.NET SDK 9.x がインストールされます。（途中で認証を要求されます）
その後、コマンドプロンプトでdotnetコマンドを認識していればインストール成功です。

テストの実行方法

dotnet コマンド経由での実行方法

  • コマンドプロンプトで\VaNillaTestProject\DxLib\project\vs2022\ディレクトリを開きます。
  • dotnet runコマンドを実行します。

Visual Studioからの実行方法

  • \VaNillaTestProject\DxLib\project\vs2022\TestProject.sln を Visual Studio 2022 で開きます。
  • ソリューション エクスプローラー上の TestProject を右クリックし、「リビルド(E)」をクリックします。
  • ビルド完了したら、ソリューション エクスプローラー上の TestProject を右クリックし、「デバッグ(G) -> 新しいインスタンスの開始(I)」をクリックします。
  • テストが実行されます。
