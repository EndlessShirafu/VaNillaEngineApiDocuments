FontResourceManager

ゲーム内における フォントリソース を扱うためのマネージャクラスと、
それにまつわるユーティリティを取り扱うモジュールです。
フォントリソースは文字の描画を行うために必要なリソースです。

FontResourceLoader

フォントリソースのロード・アンロードを行うためのユーティリティクラスです。
VaNillaエンジンの初期化後、フォントリソースはすべてロード済みの状態です。
そのまま指定のフォントリソースで文字描画を行うことが可能ですが、
メモリなどの都合でフォントのロード状態を細かく制御したい場合に本クラスを利用します。

フォントリソースのenum定義の事前生成について

フォントリソースを利用するためには、あらかじめFontResourceTypeのenum定義や、
C#上で取り扱うための情報クラスをyamlから生成しておく必要があります。
Fixture/Script/convert.pyは、利用するフォントの情報を記述したyamlから
FontResourceTypeのenum定義などが記述されたC#コード(FontResourceType.cs)を生成するためのpythonスクリプトです。
上記スクリプトでFontResourceType.csを事前生成しない場合、コンパイルエラーとなるので注意してください。

FontResourceManagerの使い方

フォントリソースは、VaNillaエンジンの初期化完了時点で自動的にメモリ上にロードされます。
基本的にはFontResourceManager.GetFontResource APIに、取得したいFontResourceTypeを指定することで
所望のフォントリソースが取得できます。
