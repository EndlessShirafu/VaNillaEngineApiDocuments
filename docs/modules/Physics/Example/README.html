Physics Example

VaNilla Physics モジュールの使用例を示すサンプルプロジェクトです。

内容

  • PhysicsDemo.cs - メインのデモシーン
  • PlayerCharacter.cs - プレイヤーキャラクター（物理制御）
  • FallingBox.cs - 落下するボックス
  • MovingPlatform.cs - 移動する足場
  • StaticFloor.cs - 静的な床
  • DebugVisualizationExample.cs - デバッグ描画の例

ビルド方法

  1. .NET 9.0 SDK がインストールされていることを確認してください
  2. build.bat を実行してビルドします
build.bat

実行方法

ビルド後、run.bat を実行するか、以下のコマンドを実行します：

cd bin\Release\net9.0-windows10.0.20348.0
PhysicsExample.exe

操作方法

  • 矢印キー - プレイヤーの移動
  • スペースキー - ジャンプ
  • F3キー - デバッグ描画の切り替え
  • Gキー - 重力の切り替え（通常/低重力）
  • ESCキー - 終了

必要な環境

  • Windows 10 以降
  • .NET 9.0 SDK
  • DirectX 9.0c 以降

注意事項

  • 初回実行時は VaNillaSettings.yaml からコード生成が必要です
  • DxLib の DLL ファイルが自動的にコピーされます
