DailySystem モジュール

DailySystemモジュールは、ゲーム内でのデイリーミッション、ログインボーナス、日次リセット機能などの日単位のゲームシステムを管理するモジュールです。 現在このモジュールは未実装です。

概要

このモジュールは以下の日次ゲームシステム機能を提供します：

  • デイリーログイン管理: プレイヤーの日次ログイン状態の追跡
  • 日時管理: UTC基準での日付・時刻管理
  • デイリーリセット: 日次でリセットされるゲーム要素の管理
  • ログイン履歴: ログイン履歴の保存と参照

使用例（予定）

// ゲーム開始時の初期化
DailyInfo.Initialize();

// ログイン処理
if (!DailyInfo.LoginedToday)
{
    DailyInfo.RecordLogin();
    
    // ログインボーナスの付与
    var todayBonus = LoginBonusManager.GetTodayBonus();
    if (todayBonus != null && !todayBonus.IsClaimed)
    {
        LoginBonusManager.ClaimBonus(todayBonus.Day);
        ShowLoginBonusDialog(todayBonus);
    }
}

// デイリーミッションの更新
DailyMissionManager.UpdateMissionProgress("kill_enemies", enemyCount);

// 日次リセットのチェック
if (DailyInfo.IsNewDay())
{
    DailyInfo.ResetDailyData();
    DailyMissionManager.ResetAllMissions();
}

注意事項

  • 時刻管理はUTC基準で行い、ローカル時刻への変換は表示時に実行
  • サーバー/クライアント時刻同期が必要な場合は別途実装を検討
  • セーブデータの整合性確保のため、適切な永続化機能との連携が必要
