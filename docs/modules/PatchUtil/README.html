PatchUtil

バイナリ差分パッチをサーバからダウンロードし、自動適用するためのユーティリティです。 VaNillaPatcher をバックエンドツールとして動作します。 （言い換えると、 VaNillaPatcher バックエンドツール類が存在していない場合、本ユーティリティは動作しません。）

機能

  • バイナリ差分パッチの載っているサーバに、新しいバージョンがあるかどうかを問い合わせる
  • 新しいバージョンが存在する場合、対象のバージョンの差分パッチをローカルディレクトリにダウンロードする
  • ダウンロードした差分パッチを適用する
  • 以上の処理を、新しいバージョンが見つからなくなるまで再帰的に実行する

API Reference

詳細なAPIドキュメントについては、VaNillaのAPIリファレンスを参照してください。

VaNilla.Patch.Util

バイナリ差分パッチの自動適用を行うユーティリティクラスです。

静的プロパティ

  • IsExecutable - 本ユーティリティが実行可能かどうか（バックエンドexe存在とネット接続確認）

静的メソッド

主要機能

  • RecursiveApply() - 再帰的なパッチ適用処理
  • QueryUpdateInformation(string serverDirectoryPath, string fileName) - 更新情報の問い合わせ

バージョン管理

  • GetNowVersionNum(string fileName) - 現在のバージョン番号を取得
  • GetNewestVersionNum(string serverDirectoryPath, string fileName) - サーバから最新バージョン番号を取得
  • RewriteVersionNum(string fileName, int newVersion) - バージョン情報を更新

ファイル操作

  • DownloadDiffFile(string serverDirectoryPath, string fileName, string dlPath) - 差分ファイルのダウンロード
  • MakeBATFile(string diffFileName) - 差分パッチ実行用バッチファイルの作成
  • CallBatch(string param = "") - バッチファイルの実行

注意点

サーバへの問い合わせ処理は同期実行されます。 通信負荷も掛かるため、ゲーム/アプリ起動直後のみサーバへ問い合わせするように実装してください。 （ゲームのメイン処理内で問い合わせを実行し続けるような使い方はしないでください）
