PatchUtil

バイナリ差分パッチをサーバからダウンロードし、自動適用するためのユーティリティです。 VaNillaPatcher をバックエンドツールとして動作します。 （言い換えると、 VaNillaPatcher バックエンドツール類が存在していない場合、本ユーティリティは動作しません。）

機能

  • バイナリ差分パッチの載っているサーバに、新しいバージョンがあるかどうかを問い合わせる
  • 新しいバージョンが存在する場合、対象のバージョンの差分パッチをローカルディレクトリにダウンロードする
  • ダウンロードした差分パッチを適用する
  • 以上の処理を、新しいバージョンが見つからなくなるまで再帰的に実行する

注意点

サーバへの問い合わせ処理は同期実行されます。 通信負荷も掛かるため、ゲーム/アプリ起動直後のみサーバへ問い合わせするように実装してください。 （ゲームのメイン処理内で問い合わせを実行し続けるような使い方はしないでください）
