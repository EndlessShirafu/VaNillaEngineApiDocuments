Moving Path Editor ユーザーマニュアル

概要

Moving Path Editorは、ゲーム内のキャラクターやオブジェクトの移動経路を視覚的に編集できるツールです。 直線や曲線を組み合わせて複雑な動きのパスを作成し、JSON形式で保存・読み込みができます。

画面構成

1. メインキャンバス（左側）

  • 白い領域: ゲーム画面内の領域
  • グレーの領域: 画面外の領域（マイナス座標を含む拡張エリア）
  • 赤い線: 原点（0,0）を示す軸線
  • グリッド: 50ピクセル単位の補助線

2. Inspector（右側）

選択中の要素の詳細設定を行うパネル

3. コマンドボタン（下部）

主要な操作ボタン

4. ステータスバー（最下部）

現在のマウス座標や操作状態を表示

基本操作

パスの作成

点を追加

  1. キャンバス上の任意の位置を左クリック
  2. 2つ目の点をクリックすると、自動的に線分（パス）が作成されます
  3. 続けてクリックすることで、連続したパスを作成できます

点を移動

  • 既存の点を左ドラッグで移動
  • ドラッグ中、ステータスバーとInspectorにリアルタイムで座標が表示されます

点を削除

  • 制御点（グレーの点）を右クリックで削除
  • パスの中間点付近をクリックで削除

パスの種類と変換

Line（直線）

  • デフォルトのパスタイプ
  • 始点と終点を直線で結ぶ

Bezier2（2次ベジェ曲線）

  • Ctrl+左ドラッグまたは中ボタンドラッグでLineから自動変換
  • 1つの制御点で曲線を調整

Bezier3（3次ベジェ曲線）

  • Inspectorから手動で設定
  • 2つの制御点でより複雑な曲線を作成

高度な編集機能

矩形選択

  1. Shift+ドラッグで複数の点を矩形選択
  2. 選択された点は黄色でハイライト表示
  3. 選択した点群をまとめて移動可能

自由変形モード

  1. 点を矩形選択
  2. 右クリック → **「自由変形」**を選択
  3. 変形ハンドルが表示される：
    • 角ハンドル: 拡大・縮小
    • 辺ハンドル: 一方向の拡大・縮小
    • 回転ハンドル: 回転
    • 中央: 移動
  4. Enterまたは右クリックで確定
  5. Escapeでキャンセル

パスの接続

  1. 2つの端点（パスの始点または終点）を選択
  2. 右クリック → 「パスを接続」
  3. 選択した端点間に新しい線分が追加されます
    • 区間削除で切れた端点も接続可能

ズーム機能

マウスホイール

  • Ctrl+ホイール上: 拡大
  • Ctrl+ホイール下: 縮小

ボタン操作

  • 拡大ボタン: 110%ずつ拡大
  • 縮小ボタン: 90%ずつ縮小
  • 100%ボタン: ズームをリセット

ズーム範囲: 10% ～ 500%

区間の設定

区間を選択

  • パスの線上をクリックで区間を選択（オレンジ色でハイライト）

区間タイプの変更

Inspectorの「区間設定」で変更：

  • Line: 直線
  • Bezier2: 2次ベジェ曲線
  • Bezier3: 3次ベジェ曲線

イージングの設定

動きの加減速を設定：

  • Linear: 等速
  • EaseIn: ゆっくり始まって加速
  • EaseOut: 減速して止まる
  • EaseInOut: ゆっくり始まってゆっくり終わる
  • EaseInQuad/OutQuad/InOutQuad: 2次関数的な加減速
  • EaseInCubic/OutCubic/InOutCubic: 3次関数的な加減速
  • EaseOutBounce: バウンド効果
  • EaseOutElastic: 弾性効果

時間の設定

  • 区間の移動にかかる時間（秒）を設定

ループ設定

ループタイプ

  • None: ループなし
  • Repeat: 指定区間を繰り返し
  • Reverse: 往復運動
  • Stop: 指定位置で停止

ループ範囲の設定

  1. 区間を選択
  2. **「ループ開始に設定」**ボタンをクリック
  3. 別の区間を選択
  4. **「ループ終了に設定」**ボタンをクリック

ループ回数

  • 0: 無限ループ
  • 1以上: 指定回数でループ

イベント送信機能

各点で特定のタイミングでイベントを発火できます：

  1. 点を選択
  2. Inspectorで**「イベント送信」**にチェック
  3. イベントパラメータ（最大4つの文字列）を入力
  4. ゲーム側でこのイベントを受け取って処理

ファイル操作

保存

  1. **「保存」**ボタンをクリック
  2. ファイル名を指定（.json形式）
  3. パス情報がJSON形式で保存されます

読込

  1. **「読込」**ボタンをクリック
  2. 保存済みのJSONファイルを選択
  3. パスが復元されます

新規作成

  1. **「新規」**ボタンをクリック
  2. 編集中のパスがある場合は確認ダイアログが表示
  3. 「はい」で新規作成、「いいえ」で編集継続

アンドゥ・リドゥ

キーボードショートカット

  • Ctrl+Z: 元に戻す（アンドゥ）
  • Ctrl+Y: やり直し（リドゥ）

ボタン操作

  • **「元に戻す」**ボタン
  • **「やり直し」**ボタン

最大50回まで操作履歴を保持

再生機能

**「再生」**ボタンをクリックすると：

  • 作成したパスに沿ってプレビュー（緑の点）が移動
  • ループ設定も反映されます
  • もう一度クリックで停止

設定項目

パス設定（Inspector）

  • 画面幅/高さ: ゲーム画面のサイズ（デフォルト: 1920×1080）
  • ループタイプ: ループの種類
  • ループ回数: 繰り返し回数
  • ループ範囲: 開始・終了区間のインデックス

点の設定（Inspector）

  • X座標/Y座標: 選択中の点の座標（-2000～6000）
  • イベント送信: イベント発火の有無
  • イベントパラメータ: 送信するパラメータ（最大4つ）

区間設定（Inspector）

  • 区間タイプ: Line/Bezier2/Bezier3
  • イージング: 動きの加減速パターン
  • 時間（秒）: 移動にかかる時間（0.01～10.0秒）

Tips

効率的な編集

  • 大まかな形を直線で作成してから、必要な部分だけ曲線に変換
  • 矩形選択で複数の点をまとめて移動・変形
  • ズーム機能を活用して細かい調整

スムーズな動きの作成

  • イージングを使って自然な加減速を演出
  • Bezier曲線で滑らかなカーブを作成
  • 時間設定で速度を調整

複雑なパスの作成

  • ループ機能で繰り返し動作を簡単に実現
  • イベント送信で特定タイミングでの処理を実装
  • 画面外の座標も使用可能（-2000～6000の範囲）

トラブルシューティング

Q: 点が追加できない

A: 既存の点や線の近くをクリックしていないか確認してください。空いている場所をクリックしてください。

Q: スクロール位置がリセットされる

A: 最新バージョンでは修正されています。問題が続く場合はアプリケーションを再起動してください。

Q: パスが表示されない

A: 最低2つの点が必要です。1つ目の点を配置した後、2つ目の点をクリックしてください。

Q: 変形モードから抜けられない

A: Escapeキーを押すか、右クリックで確定/キャンセルできます。

Q: ループが機能しない

A: ループ開始・終了区間が正しく設定されているか、Inspectorで確認してください。

データ形式

保存されるJSONファイルには以下の情報が含まれます：

  • 各区間の始点・終点座標
  • 区間タイプ（Line/Bezier2/Bezier3）
  • 制御点の座標
  • イージングタイプ
  • 各区間の時間
  • ループ設定
  • イベント情報
  • 画面サイズ

このデータはゲームエンジン側で読み込んで、実際のキャラクター移動に使用できます。
