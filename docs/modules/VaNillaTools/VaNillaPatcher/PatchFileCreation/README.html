PatchFileCreation

PatchFileCreation ツールは、パッチを作成するためのツールです。 本ディレクトリ内のツールは、すべてパッチを更新する開発者向けのツールのため、 ゲームやアプリのエンドユーザーが直接触れることはありません。

パッチ作成手順

  1. EncryptedVersionFileMaker で、バージョン番号を暗号化したバージョン番号ファイル version.dat を作成します。 この version.dat の中身には、配布したいパッチのバージョンを記述します。 (例: 配りたいパッチのバージョンがv1.02ならば「102」を入力)
  2. 上記で生成した version.dat をゲーム内のフォルダの任意の位置に配置します。
  3. DirectoryPatchMaker で、パッチ更新前のディレクトリと、パッチ更新後のディレクトリを指定します。 ここで、 version.dat も更新の対象に含めるようにします。
  4. DirectoryPatchMaker で生成したパッチを、http アクセスが可能なサーバーに配置します。
  5. サーバーに配置してある version.dat を、最新の version.dat で書き換えます。

サブディレクトリについて

上記「パッチ作成手順」の各工程を個別のコマンドラインツールに分離してあります。 また、GUIツールも存在します。

DirectoryPatchMaker

2つのディレクトリを入力とし、差分をバイナリパッチとして作成するツールです。

EncryptedVersionFileMaker

入力した3桁のバージョン文字列を、入力したパスワードを用いて暗号化し、ファイルとして出力します。 上記によって出力されたバージョン文字列ファイルの暗号化を解除するための version_file_decrypter も同梱しています。

FrontEnd

DirectoryPatchMaker と EncryptedVersionFileMaker の処理を両方実行するためのツールです。

PatchMakerGui

FrontEnd の処理をGUI経由で実行できるツールです。
