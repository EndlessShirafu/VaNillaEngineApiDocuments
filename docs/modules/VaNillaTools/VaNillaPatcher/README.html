VaNillaPatcher

これはなに？

バイナリパッチ差分の作成ツール + パッチの適用ツールです。 主に、ゲーム起動時のデータの自動更新などに利用できます。

サポートしている機能

  • ディレクトリ単位でのパッチ差分の作成
  • パッチバイナリの適用

ディレクトリ構成について

基本的にゲーム開発者は Deploy/PatchMaker と Deploy/Patcher ディレクトリのみ利用すればOKです。

パッチツール自体を改造したい場合には PatchApply や PatchFileCreation ディレクトリを確認してください。 上記ディレクトリに、ツールのソースコードが配置されています。 Deploy/deploy.bat には各ツールをビルドし、 Deploy ディレクトリ以下に配置するためのコマンド群が書かれています。

開発環境

  • Python 3.6.2
  • 必要なPyPIパッケージは、各サブディレクトリの requirements.txt に記載してあります。

使い方

  1. Deploy/Patcher の中身を配布するゲームのディレクトリに含めておきます。
