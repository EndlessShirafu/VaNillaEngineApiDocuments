VaNilla Launcher

VaNillaエンジンで作成されたプロジェクトを管理するためのランチャーアプリケーション。Unity Hubと同様の操作性を提供します。

機能

  • プロジェクト管理

    • VaNillaプロジェクトの一覧表示
    • プロジェクトの追加・削除
    • お気に入り機能
    • プロジェクト検索

  • プロジェクト起動

    • VaNillaStudioとVisual Studio 2026の同時起動
    • エクスプローラーで開く

  • プロジェクト作成

    • テンプレートからの新規プロジェクト作成
    • 標準テンプレート（空のプロジェクト、2D/3Dゲーム基本）

  • 表示モード

    • リストビュー
    • グリッドビュー
    • コンパクトビュー

要件

  • Windows 11
  • .NET 10.0 Runtime
  • Visual Studio 2026（プロジェクト開発用）

ビルド方法

# リポジトリをクローン
git clone [repository-url]
cd VaNillaLauncher

# ビルド
dotnet build

# 実行
dotnet run

開発環境

  • C# 14.0
  • .NET 10.0
  • Avalonia UI 11.2.2
  • CommunityToolkit.Mvvm 8.3.2

プロジェクト構造

VaNillaLauncher/
├── Models/              # データモデル
├── ViewModels/          # MVVM ViewModels
├── Views/               # UIビュー
├── Services/            # ビジネスロジック
├── Converters/          # 値コンバーター
├── Assets/              # リソースファイル
└── Resources/           # アプリケーションリソース

設定ファイル

設定ファイルは以下の場所に保存されます：

  • Windows: %APPDATA%\VaNillaLauncher\

settings.json

ランチャーの設定（テーマ、パス、ウィンドウ状態など）

projects.json

管理しているプロジェクトのリスト
