NineSliceRenderer

9スライスを表示するためのExpresserComponentです。

APIリファレンス

詳細なAPIドキュメントはVaNilla NineSliceRenderer APIリファレンスをご覧ください。

用途と特徴

ウィンドウUI等に利用される9スライスを表示するためのExpresserComponentを提供します。

主な機能

  • 9スライスの表示
  • 回転とスケーリング

サンプル

//NineSliceRendererはHasTransformがrequiredです
[AutoGenerateConstructor]
[DefaultExpresserComponent(typeof(Transform.HasTransform), auto_member_generate: true)]
[DefaultExpresserComponent(typeof(Renderer.NineSliceRenderer), auto_member_generate: true)]
public partial class NineSliceSample : Object {
    public override bool SetUp() {
        hasTransform.hasPosition.Position = new Transform.Point2(300, 300);
        
        // 幅と高さ、スライスする位置の設定
        nineSliceRenderer.SetSlicePosition(300, 300, 100, 50, 100, 50);

        for (int i = 0; i < nineSliceRenderer.SlicedTexture.Length; i++) {
            nineSliceRenderer.SlicedTexture[i] = /* テクスチャ割り当て */;
        }

        return true;
    }

    public override void TearDown() {
    }
}
