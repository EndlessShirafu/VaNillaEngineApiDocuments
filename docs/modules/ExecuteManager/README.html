ExecuteManager

各種ExecuteのManagerです。 Executeの停止/再開や、呼び出し頻度の変更等を変更します。

IExecuter

ExcuteManagerで管理するモノが実装するインターフェースです。 以下これを実装しているモノをExecuterと呼びます。

ExecuteType

実行の種類を指定します。将来的にはユーザーが自由に定義出来る想定かつ現状も変更される事によって不具合はないはずですが、現状は VaNilla 側のソースコード上で定義しています。 値が0のもののみdefaultになるので注意してください(C#のenumのdefault仕様)。

使用例

VaNilla.Execute.ExecuteManager.Pause();                     //(デフォルトのExecuteTypeを持つものを) 停止
VaNilla.Execute.ExecuteManager.Pause(ExecuteType.System);   //Systemを持つものを停止
VaNilla.Execute.ExecuteManager.Resume();                    //(デフォルトのExecuteTypeを持つものを) 再開
VaNilla.Execute.ExecuteManager.Pause(ExecuteType.System);   //Systemを持つものを再開
