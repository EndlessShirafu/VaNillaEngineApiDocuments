CodeModifier

コードの文字コードや改行コードを修正するプログラム。 VaNillaを構成するコードではなく、VaNilla運用上のツール。 現状は 文字コードはUTF-8 with BOM、改行コードはLFとなっているのでそれに適合するように修正する。

プログラムの構成

現状各ファイルの文字コードの識別はnkfをラップしたライブラリであるNkf.Netを使用している。 これをCodeModifierのNuGetに追加している

Nkf.Netの不具合

Nkf.NetのReaderは使用すると改行コードをCRLFに強制変換してしまう。 また、行末に改行があった場合にそれを削除してしまう。

https://github.com/kkato233/Nkf.Net/issues/7

で製作者の方に連絡したところファイル末尾については修正予定だが、行末については現状仕様だそう。

よって自作のNkfTextReaderExクラスを作成してこの問題を回避する。

バイナリ

現状アップされるバイナリとしては VaNilla\CodeModifier\CodeModifier\CodeModifier\bin\Release 以下のものを想定しているのでこれを使用する

Nkf.Netソリューション

Nkf.Netフォルダ以下のソリューションはNkf.Net側に手を入れてdllを出力しCodeModifier側のdllをそれに差し替えて上記問題を解決するのに使用していた。 上記の方法で解決するに至ったので現状は必要ないが、処理を書く際に参考になることがあるので一旦残してある。
