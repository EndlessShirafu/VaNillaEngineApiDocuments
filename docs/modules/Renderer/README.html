Renderer

VaNillaエンジンの中核となる描画システムモジュールです。プラットフォームに依存しない統一されたAPIを通じて、2D/3Dグラフィックスの描画を行います。

概要

Rendererモジュールは、VaNillaエンジンの描画哲学である「明示的描画」を実現する静的APIを提供します。従来のゲームエンジンのようなオブジェクトベースの自動描画ではなく、開発者が描画タイミングと内容を完全にコントロールできる設計になっています。

APIリファレンス

詳細なAPIドキュメントはVaNilla Renderer APIリファレンスをご覧ください。

アーキテクチャ

Renderer/
├── Interface/          # プラットフォーム非依存のAPI定義
│   ├── Utils.cs       # 主要な描画API (静的メソッド群)
│   ├── DrawObject.cs  # 描画オブジェクトパターン実装
│   ├── Color3.cs      # RGB色構造体
│   └── Color4.cs      # RGBA色構造体
└── Implementation/     # プラットフォーム固有実装
    ├── DxLib/         # DxLibバックエンド
    └── Unity/         # Unityバックエンド

設計思想

明示的描画 (Explicit Rendering)

VaNillaエンジンでは、「描画命令を明示的に発行した場合だけ、描画を行う」という考え方を採用しています。

従来のゲームエンジン:

  • オブジェクトにマテリアルを割り当て
  • レンダラーコンポーネントを追加
  • エンジンが自動的に描画

VaNillaエンジン:

  • 描画タイミングを完全制御
  • Objectの存在は不要
  • 描画命令の直接発行

この設計により、以下が可能になります：

  • 複雑な描画ロジックの実装
  • 動的な描画内容の変更
  • パフォーマンスの最適化
  • 特殊なエフェクトの実現

主要機能

1. テクスチャ描画

基本的な描画から高度な描画まで、豊富なオーバーロードを提供：

// シンプルな描画
Utils.Draw(x, y, textureType);

// 詳細なパラメータ指定
Utils.Draw(
    x: 100, y: 200,
    texture_type: TextureResourceType.Player,
    origin: DrawOrigin.Center,
    shader: ShaderManager.ShaderEnum.ALPHA,
    rot_radian: MathUtil.DegreeToRadian(45),
    extend_rate_x: 2.0f, extend_rate_y: 2.0f,
    red: 1.0f, green: 0.5f, blue: 0.5f, alpha: 0.8f,
    reverse_x: false, reverse_y: true
);

// 色構造体を使用した描画
Utils.Draw(x, y, texture, new Color4(1, 0, 0, 0.5f));

2. 高度な描画機能

任意軸回転描画

Utils.DrawRot(x, y, texture,
    rot_radian_x: 0.5f,
    rot_radian_y: 1.0f,
    rot_radian_z: 0.3f
);

4点座標指定描画

// 自由変形描画
Utils.Draw(
    x0, y0,  // 左上
    x1, y1,  // 右上
    x2, y2,  // 右下
    x3, y3,  // 左下
    textureType
);

バッチ描画

// 大量のスプライトを効率的に描画
Point2[] positions = /* ... */;
Point2[] uvs = /* ... */;
Utils.Draw(positions, uvs, texture,
    primitive_type: DrawPrimitiveType.TriangleList
);

3. プリミティブ描画

// 矩形
Utils.DrawBox(x, y, width, height,
    origin: DrawOrigin.Center,
    fill_flag: true,
    color: Color4.red
);

// 円
Utils.DrawCircle(centerX, centerY, radius,
    fill_flag: false,  // 輪郭のみ
    color: new Color4(0, 1, 0, 0.5f)
);

// 線
Utils.DrawLine(startX, startY, endX, endY,
    thickness: 3.0f,
    color: Color4.blue
);

// 三角形
Utils.DrawTriangle(x0, y0, x1, y1, x2, y2,
    fill_flag: true,
    color: Color4.white
);

4. 文字列描画

// 基本的な文字列描画
Utils.DrawString(x, y, "Hello VaNilla!",
    font_type: FontResourceType.Default
);

// ルビ付き日本語描画
Utils.DrawString(x, y, "振り仮名|ふりがな|付きテキスト",
    font_type: FontResourceType.Japanese,
    analyze_japanese_ruby: true,
    japanese_ruby_font: rubyFont
);

5. 特殊効果

円形ゲージ

// HPバーやタイマーなどに使用
Utils.DrawCircleGauge(x, y, gaugeTexture,
    gauge_percentage: 0.75f,    // 75%表示
    start_percentage: 0.0f,     // 開始位置
    extend_rate: 1.5f
);

描画パラメータ詳細

DrawOrigin (描画原点)

  • UpLeft: 左上を原点として描画
  • Center: 中央を原点として描画

ShaderEnum (合成モード)

  • ALPHA: アルファブレンド（標準）
  • ADD: 加算合成
  • MULTIPLY: 乗算合成
  • SUBTRACT: 減算合成
  • SCREEN: スクリーン合成
  • INVSRC: 反転合成

色指定

  • Color3: RGB値（0.0～1.0）
  • Color4: RGBA値（0.0～1.0）
  • 定義済み色定数: Color4.red, Color4.green, Color4.blue, Color4.black, Color4.white

使用例

基本的なゲームオブジェクトの描画

[AutoGenerateConstructor]
[DefaultExpresserComponent(typeof(Transform.HasPosition), auto_member_generate: true)]
[DefaultExpresserComponent(typeof(PlayerRenderComponent), auto_member_generate: true)]
public partial class Player : VaNilla.Object
{
    public override bool SetUp()
    {
        hasPosition.Position = new Point2(320, 240);
        return true;
    }
}

[RequiredExpresserComponent(typeof(Transform.HasPosition), auto_member_generate: true)]
public partial class PlayerRenderComponent : IExpresserComponent
{
    public bool Active { get; set; }
    public Execute.ExecuteType ExecuteType { get; set; }
    
    private float currentHp = 100f;
    private float maxHp = 100f;

    public bool Execute(Object associated_object, in Time.Time current_time)
    {
        // プレイヤーの描画
        Utils.Draw(
            hasPosition.Position.X, hasPosition.Position.Y,
            TextureResourceType.Player,
            origin: DrawOrigin.Center
        );
        
        // HPバーの描画
        float hpRatio = currentHp / maxHp;
        Utils.DrawBox(
            hasPosition.Position.X - 50, hasPosition.Position.Y - 60,
            100 * hpRatio, 10,
            origin: DrawOrigin.UpLeft,
            color: new Color4(1 - hpRatio, hpRatio, 0)
        );
        
        return true;
    }
    
    public bool SetUp(Object associated_object) { return true; }
    public void TearDown(Object associated_object) { }
    public void HoldAssociatedObject(Object associated_object) { }
    public void ReleaseAssociatedObject(Object associated_object) { }
    public void GetComponentForSetUp(Object associated_object) { }
    public void ReleaseComponentForTearDown(Object associated_object) { }
}

エフェクトの描画

[AutoGenerateConstructor]
[DefaultExpresserComponent(typeof(ParticleEffectComponent), auto_member_generate: true)]
public partial class ParticleEffect : VaNilla.Object
{
    public override bool SetUp()
    {
        return true;
    }
}

public class ParticleEffectComponent : IExpresserComponent
{
    public bool Active { get; set; }
    public Execute.ExecuteType ExecuteType { get; set; }
    
    private List<Particle> particles = new List<Particle>();

    public bool Execute(Object associated_object, in Time.Time current_time)
    {
        // 加算合成で光るエフェクト
        foreach (var particle in particles)
        {
            Utils.Draw(
                particle.X, particle.Y,
                TextureResourceType.Particle,
                shader: ShaderManager.ShaderEnum.ADD,
                alpha: particle.Life,
                extend_rate_x: particle.Scale,
                extend_rate_y: particle.Scale
            );
        }
        
        return true;
    }
    
    public bool SetUp(Object associated_object) { return true; }
    public void TearDown(Object associated_object) { }
    public void HoldAssociatedObject(Object associated_object) { }
    public void ReleaseAssociatedObject(Object associated_object) { }
    public void GetComponentForSetUp(Object associated_object) { }
    public void ReleaseComponentForTearDown(Object associated_object) { }
}

動的な描画パラメータ制御

[AutoGenerateConstructor]
[DefaultExpresserComponent(typeof(Transform.HasPosition), auto_member_generate: true)]
[DefaultExpresserComponent(typeof(IsChangeableAlpha), auto_member_generate: true)]
[DefaultExpresserComponent(typeof(IsChangeableColor), auto_member_generate: true)]
[DefaultExpresserComponent(typeof(DynamicRenderComponent), auto_member_generate: true)]
public partial class FadingObject : Object
{
    public override bool SetUp()
    {
        hasPosition.Position = new Point2(100, 100);
        isChangeableAlpha.Alpha = 1.0f;
        isChangeableColor.Color = new Color3(1, 1, 1);
        return true;
    }
}

[RequiredExpresserComponent(typeof(Transform.HasPosition), auto_member_generate: true)]
[RequiredExpresserComponent(typeof(IsChangeableAlpha), auto_member_generate: true)]
[RequiredExpresserComponent(typeof(IsChangeableColor), auto_member_generate: true)]
public partial class DynamicRenderComponent : IExpresserComponent
{
    public bool Active { get; set; } = true;
    public Execute.ExecuteType ExecuteType { get; set; } = Execute.ExecuteType.Render;
    
    public bool Execute(Object associated_object, in Time.Time current_time)
    {
        // フェードアウト効果
        isChangeableAlpha.Alpha -= 0.01f * current_time.DeltaTime;
        if (isChangeableAlpha.Alpha <= 0) isChangeableAlpha.Alpha = 0;
        
        // 色の変化
        isChangeableColor.Color = new Color3(
            MathF.Sin(current_time.TotalTime) * 0.5f + 0.5f,
            MathF.Cos(current_time.TotalTime) * 0.5f + 0.5f,
            1.0f
        );
        
        // 描画
        Utils.Draw(
            hasPosition.Position.X, hasPosition.Position.Y,
            TextureResourceType.TestSprite,
            origin: BasicRenderer.DrawOrigin.Center,
            red: isChangeableColor.Color.R,
            green: isChangeableColor.Color.G,
            blue: isChangeableColor.Color.B,
            alpha: isChangeableAlpha.Alpha
        );
        
        return true;
    }
    
    public bool SetUp(Object associated_object) { return true; }
    public void TearDown(Object associated_object) { }
    public void HoldAssociatedObject(Object associated_object) { }
    public void ReleaseAssociatedObject(Object associated_object) { }
    public void GetComponentForSetUp(Object associated_object) { }
    public void ReleaseComponentForTearDown(Object associated_object) { }
}

注意事項

  1. 描画順序: 描画命令は呼び出し順序で実行されます
  2. 座標系: スクリーン座標系を使用（左上が原点）
  3. 回転: ラジアン単位で指定
  4. 色値: 0.0～1.0の範囲で指定（0～255ではない）
  5. パフォーマンス: バッチ描画APIを使用することで、大量のスプライト描画を最適化可能
