Easing モジュール

Easingモジュールは、ゲーム開発で欠かせないイージング（補間）機能を提供するモジュールです。リニアからバウンス、エラスティックまで、幅広いイージング関数を用意し、滑らかで自然なアニメーションを簡単に実装できます。

概要

このモジュールは以下の機能を提供します：

  • 包括的なイージング関数: Linear、Quadratic、Cubic、Quartic、Quintic、Sinusoidal、Exponential、Circular、Bounce、Back、Elastic
  • In/Out/InOut対応: 各イージングタイプでEaseIn、EaseOut、EaseInOut をサポート
  • Range型統合: VaNilla.Math.Rangeとの連携で型安全な範囲指定
  • 時間ベース補間: 時刻、最小値、最大値、期間を指定した補間計算
  • 高性能実装: 静的クラスによる効率的な関数提供

実装状況

使用方法

基本的なイージング

// 2秒間で0から100まで二次関数でイージング
float currentTime = 1.0f;  // 現在時刻
float duration = 2.0f;     // 継続時間
float startValue = 0.0f;   // 開始値
float endValue = 100.0f;   // 終了値

float easedValue = Easing.Util.EaseInQuadratic(currentTime, startValue, endValue, duration);
// 結果: 25.0f (1秒時点での二次関数イージング値)

Range型を使用したイージング

// Range型を使用した型安全なイージング
var range = new Math.Range(0.0, 100.0);
float easedValue = Easing.Util.EaseOutCubic(1.5f, range, 2.0f);

アニメーションでの使用例

public class EasedMovement : IExpresserComponent
{
    private HasPosition position;
    private float animationTime = 0.0f;
    private float animationDuration = 2.0f;
    
    public Vector2 StartPosition { get; set; }
    public Vector2 EndPosition { get; set; }
    public bool IsAnimating { get; private set; } = false;
    
    public bool SetUp(Object object__)
    {
        position = object__.GetExpresserComponent<HasPosition>();
        return true;
    }
    
    public bool Execute(Object object__, in Time current_time)
    {
        if (IsAnimating)
        {
            animationTime += current_time.DeltaTime;
            
            if (animationTime >= animationDuration)
            {
                position.Position = EndPosition;
                IsAnimating = false;
                animationTime = 0.0f;
            }
            else
            {
                // X座標とY座標をそれぞれイージング
                float easedX = Easing.Util.EaseInOutQuadratic(
                    animationTime, StartPosition.X, EndPosition.X, animationDuration);
                float easedY = Easing.Util.EaseInOutQuadratic(
                    animationTime, StartPosition.Y, EndPosition.Y, animationDuration);
                
                position.Position = new Vector2(easedX, easedY);
            }
        }
        
        return true;
    }
    
    public void StartAnimation(Vector2 start, Vector2 end)
    {
        StartPosition = start;
        EndPosition = end;
        animationTime = 0.0f;
        IsAnimating = true;
    }
    
    public void TearDown(Object object__) { }
    public bool Active { get; set; } = true;
    public Execute.ExecuteType ExecuteType { get; set; } = Execute.ExecuteType.Update;
}

UI要素のフェードイン/フェードアウト

public class FadeEffect : IExpresserComponent
{
    private HasAlpha alpha;
    private float fadeTime = 0.0f;
    private float fadeDuration = 1.0f;
    private bool isFadingIn = false;
    private bool isFadingOut = false;
    
    public bool SetUp(Object object__)
    {
        alpha = object__.GetExpresserComponent<HasAlpha>();
        return true;
    }
    
    public bool Execute(Object object__, in Time current_time)
    {
        if (isFadingIn)
        {
            fadeTime += current_time.DeltaTime;
            if (fadeTime >= fadeDuration)
            {
                alpha.Alpha = 1.0f;
                isFadingIn = false;
                fadeTime = 0.0f;
            }
            else
            {
                alpha.Alpha = Easing.Util.EaseOutCubic(fadeTime, 0.0f, 1.0f, fadeDuration);
            }
        }
        else if (isFadingOut)
        {
            fadeTime += current_time.DeltaTime;
            if (fadeTime >= fadeDuration)
            {
                alpha.Alpha = 0.0f;
                isFadingOut = false;
                fadeTime = 0.0f;
            }
            else
            {
                alpha.Alpha = Easing.Util.EaseInCubic(fadeTime, 1.0f, 0.0f, fadeDuration);
            }
        }
        
        return true;
    }
    
    public void FadeIn(float duration = 1.0f)
    {
        fadeDuration = duration;
        fadeTime = 0.0f;
        isFadingIn = true;
        isFadingOut = false;
    }
    
    public void FadeOut(float duration = 1.0f)
    {
        fadeDuration = duration;
        fadeTime = 0.0f;
        isFadingOut = true;
        isFadingIn = false;
    }
    
    public void TearDown(Object object__) { }
    public bool Active { get; set; } = true;
    public Execute.ExecuteType ExecuteType { get; set; } = Execute.ExecuteType.Update;
}

イージングタイプ別の特徴

Linear（リニア）

// 一定速度での補間
float value = Easing.Util.Linear(time, 0.0f, 100.0f, 2.0f);
  • 用途: 一定速度での移動、回転、スケール変更
  • 特徴: 等速直線運動、機械的な動き

Quadratic（二次関数）

// 緩やかな加速・減速
float valueIn = Easing.Util.EaseInQuadratic(time, 0.0f, 100.0f, 2.0f);
float valueOut = Easing.Util.EaseOutQuadratic(time, 0.0f, 100.0f, 2.0f);
float valueInOut = Easing.Util.EaseInOutQuadratic(time, 0.0f, 100.0f, 2.0f);
  • 用途: 自然な加速・減速、UI要素の移動
  • 特徴: 滑らかで自然な動き

Cubic（三次関数）

// より強い加速・減速
float value = Easing.Util.EaseInOutCubic(time, 0.0f, 100.0f, 2.0f);
  • 用途: メニューのスライドイン、カードの移動
  • 特徴: はっきりとした加速・減速感

Bounce（バウンス）

// 弾む動き
float value = Easing.Util.EaseOutBounce(time, 0.0f, 100.0f, 2.0f);
  • 用途: オブジェクトの着地、ボタンのプレス効果
  • 特徴: 楽しくダイナミックな動き

Back（オーバーシュート）

// 目標を少し超えてから戻る
float s = 1.70158f; // オーバーシュートの強さ
float value = Easing.Util.EaseOutBack(time, 0.0f, 100.0f, 2.0f, s);
  • 用途: UI要素の強調、ポップアップの表示
  • 特徴: 注意を引く動き

Elastic（エラスティック）

// ゴムのような弾性
float amplitude = 1.0f;  // 振幅
float period = 0.3f;     // 周期
float value = Easing.Util.EaseOutElastic(time, 0.0f, 100.0f, 2.0f, amplitude, period);
  • 用途: 通知の強調、特殊効果
  • 特徴: 目を引く振動的な動き

ゲーム開発での実用例

プレイヤーのジャンプ動作

public class JumpController : IExpresserComponent
{
    private HasPosition position;
    private float jumpTime = 0.0f;
    private float jumpDuration = 0.8f;
    private float groundY;
    private float jumpHeight = 50.0f;
    private bool isJumping = false;
    
    public bool SetUp(Object object__)
    {
        position = object__.GetExpresserComponent<HasPosition>();
        groundY = position.Y;
        return true;
    }
    
    public bool Execute(Object object__, in Time current_time)
    {
        if (isJumping)
        {
            jumpTime += current_time.DeltaTime;
            
            if (jumpTime >= jumpDuration)
            {
                position.Y = groundY;
                isJumping = false;
                jumpTime = 0.0f;
            }
            else
            {
                // 放物線を描くジャンプ
                float height = Easing.Util.EaseInOutQuadratic(
                    jumpTime, 0.0f, jumpHeight, jumpDuration * 0.5f);
                
                if (jumpTime > jumpDuration * 0.5f)
                {
                    float fallTime = jumpTime - (jumpDuration * 0.5f);
                    height = Easing.Util.EaseInQuadratic(
                        fallTime, jumpHeight, 0.0f, jumpDuration * 0.5f);
                }
                
                position.Y = groundY + height;
            }
        }
        
        return true;
    }
    
    public void Jump()
    {
        if (!isJumping)
        {
            jumpTime = 0.0f;
            isJumping = true;
        }
    }
}

カメラの滑らかなズーム

public class CameraZoom : IExpresserComponent
{
    private HasScale scale;
    private float zoomTime = 0.0f;
    private float zoomDuration = 1.5f;
    private float startZoom;
    private float targetZoom;
    private bool isZooming = false;
    
    public bool SetUp(Object object__)
    {
        scale = object__.GetExpresserComponent<HasScale>();
        return true;
    }
    
    public bool Execute(Object object__, in Time current_time)
    {
        if (isZooming)
        {
            zoomTime += current_time.DeltaTime;
            
            if (zoomTime >= zoomDuration)
            {
                scale.Scale = new Vector2(targetZoom, targetZoom);
                isZooming = false;
                zoomTime = 0.0f;
            }
            else
            {
                float currentZoom = Easing.Util.EaseInOutSinusoidal(
                    zoomTime, startZoom, targetZoom, zoomDuration);
                scale.Scale = new Vector2(currentZoom, currentZoom);
            }
        }
        
        return true;
    }
    
    public void ZoomTo(float zoom, float duration = 1.5f)
    {
        startZoom = scale.Scale.X;
        targetZoom = zoom;
        zoomDuration = duration;
        zoomTime = 0.0f;
        isZooming = true;
    }
}

パーティクルシステムとの連携

public class ParticleLifetime : IExpresserComponent
{
    private HasAlpha alpha;
    private HasScale scale;
    private float lifetime = 0.0f;
    private float maxLifetime = 3.0f;
    
    public bool SetUp(Object object__)
    {
        alpha = object__.GetExpresserComponent<HasAlpha>();
        scale = object__.GetExpresserComponent<HasScale>();
        return true;
    }
    
    public bool Execute(Object object__, in Time current_time)
    {
        lifetime += current_time.DeltaTime;
        
        if (lifetime >= maxLifetime)
        {
            object__.Destroy();
            return false;
        }
        
        // フェードアウトとスケールダウン
        float fadeProgress = lifetime / maxLifetime;
        
        // アルファ値を指数関数的に減少
        alpha.Alpha = Easing.Util.EaseOutExponential(
            fadeProgress, 1.0f, 0.0f, 1.0f);
        
        // スケールを円関数で縮小
        float scaleValue = Easing.Util.EaseInCircular(
            fadeProgress, 1.0f, 0.0f, 1.0f);
        scale.Scale = new Vector2(scaleValue, scaleValue);
        
        return true;
    }
}

高度な活用例

カスタムイージング合成

public static class CustomEasing
{
    // 複数のイージング関数を組み合わせた独自の効果
    public static float ShakeEasing(float time, float min, float max, float duration)
    {
        float baseValue = Easing.Util.EaseOutBounce(time, min, max, duration);
        
        // 高周波の振動を追加
        float shake = Easing.Util.EaseOutExponential(time, 5.0f, 0.0f, duration) *
                      Math.Util.Sin(time * 50.0f);
        
        return baseValue + shake;
    }
    
    // 段階的なイージング
    public static float SteppedEasing(float time, float min, float max, float duration, int steps)
    {
        float stepSize = duration / steps;
        int currentStep = (int)(time / stepSize);
        float stepProgress = (time % stepSize) / stepSize;
        
        float stepMin = min + (max - min) * currentStep / steps;
        float stepMax = min + (max - min) * (currentStep + 1) / steps;
        
        return Easing.Util.EaseInOutQuadratic(stepProgress, stepMin, stepMax, 1.0f);
    }
}

イージング選択システム

public enum EasingType
{
    Linear,
    QuadraticIn, QuadraticOut, QuadraticInOut,
    CubicIn, CubicOut, CubicInOut,
    BounceOut,
    BackOut,
    ElasticOut
}

public static class EasingSelector
{
    public static float ApplyEasing(EasingType type, float time, float min, float max, float duration)
    {
        return type switch
        {
            EasingType.Linear => Easing.Util.Linear(time, min, max, duration),
            EasingType.QuadraticIn => Easing.Util.EaseInQuadratic(time, min, max, duration),
            EasingType.QuadraticOut => Easing.Util.EaseOutQuadratic(time, min, max, duration),
            EasingType.QuadraticInOut => Easing.Util.EaseInOutQuadratic(time, min, max, duration),
            EasingType.CubicIn => Easing.Util.EaseInCubic(time, min, max, duration),
            EasingType.CubicOut => Easing.Util.EaseOutCubic(time, min, max, duration),
            EasingType.CubicInOut => Easing.Util.EaseInOutCubic(time, min, max, duration),
            EasingType.BounceOut => Easing.Util.EaseOutBounce(time, min, max, duration),
            EasingType.BackOut => Easing.Util.EaseOutBack(time, min, max, duration, 1.70158f),
            EasingType.ElasticOut => Easing.Util.EaseOutElastic(time, min, max, duration, 1.0f, 0.3f),
            _ => Easing.Util.Linear(time, min, max, duration)
        };
    }
}

パフォーマンス考慮事項

  • 効率的な計算: 数学関数の最適化により高速な計算
  • メモリ効率: 静的関数による最小限のメモリ使用
  • 時間精度: float精度での滑らかな補間
  • スケーラビリティ: 大量のオブジェクトでの同時アニメーション対応

注意事項

時間パラメータ

  • time: 現在の経過時間（0以上）
  • duration_time: アニメーション継続時間（正の値）
  • 時間が期間を超えた場合は最終値が返される

値の範囲

  • min、max: 最小値・最大値（任意の浮動小数点値）
  • 結果は常に[min, max]の範囲内に収まる
  • min > maxでも正常に動作（逆方向補間）

特殊パラメータ

  • Back: sパラメータでオーバーシュートの強さを調整（通常1.70158f）
  • Elastic: a（振幅）、p（周期）で弾性の特性を調整

依存関係

  • MathUtil - Math.Util.Clamp、Math.Util.PI、三角関数等
  • System.Math - .NET標準の数学関数

関連モジュール

  • TimeManager - 時間管理
  • Transform - 座標・回転・スケール変換
  • Animation - アニメーションシステム
  • UI - UIアニメーション

このモジュールにより、ゲームに自然で魅力的な動きを簡単に追加できます。
