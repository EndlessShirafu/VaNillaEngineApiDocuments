TextInput Unit Tests

このプロジェクトはVaNillaのTextInputシステムの単体テストです。

テストの実行方法

# プロジェクトディレクトリに移動
cd /mnt/d/temp/VaNillaClaude/VaNilla/TextInput/Test/TextInputTest

# テストを実行
dotnet test

テストクラス

TextInputLoggerTests

  • TextInputLoggerクラスの基本機能をテスト
  • コンストラクタ、Start/Stop、テキスト操作、カーソル位置、選択範囲など

TextInputManagerTests

  • TextInputManagerクラスのGUI統合機能をテスト
  • コントロールとの連携、複数コントロール管理、コンテキスト管理など

TextInputSettingsTests

  • TextInputSettings構造体と入力モードをテスト
  • デフォルト設定、カスタム設定、各種入力モードの確認

TextInputNativeTests

  • ネイティブレイヤーの機能をテスト
  • 文字入力可否判定、クリップボード操作、IME位置設定など

注意事項

  • これらのテストはDxLib実装に依存しない部分のみをテストしています
  • 実際のキー入力やIME動作はモックされています
  • GUI環境が必要なテストは含まれていません
