IsLoaded(SoundInfo)

指定されたサウンドリソースのロード完了状態を確認します。

public static bool IsLoaded(SoundInfo info)

Parameters

info SoundInfo

ロード状態を確認するサウンド情報

Returns

bool
  • true: サウンドリソースのロードが完了し、再生可能状態
  • false: ロード中、またはロード未開始状態

Remarks

このメソッドはLoadメソッド呼び出し後のロード進行状況をポーリングするために使用します。

ロードが完了していないサウンドを再生しようとすると警告ログが出力されます。

Load(SoundInfo)

指定されたサウンド情報を基にサウンドリソースの非同期ロードを開始します。

public static SoundResource Load(SoundInfo info)

Parameters

info SoundInfo

ロードするサウンドの情報（ファイルパス、リソース種別など）

Returns

SoundResource

ロード中のサウンドリソースハンドル。ロード完了前は無効値の状態

Remarks

このAPIは即座に復帰し、バックグラウンドでロード処理を実行します。

即座にサウンドが再生可能になるわけではありません。ロード完了までに数フレームから数秒かかる場合があります。

ロードの進行状況を確認するにはIsLoaded(SoundInfo)メソッドを使用してください。

ロード失敗時はエラーログが出力され、無効なリソースが返されます。

Unload(SoundInfo)

指定されたサウンドリソースをメモリから解放します。

public static void Unload(SoundInfo info)

Parameters

info SoundInfo

解放するサウンド情報

Remarks

メモリ使用量を削減するため、使用しないサウンドリソースは積極的に解放してください。

再生中のサウンドを解放すると、音声が即座に停止します。

解放後に再度使用する場合は、再度Loadメソッドを呼び出す必要があります。
