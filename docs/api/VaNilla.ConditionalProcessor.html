指定した時間が経過したら自動的にオブジェクトを破棄するExpresserComponentです。 カウントダウンタイマーとして機能し、指定時間後にオブジェクトの自動破棄を実行します。

DesiredExecution

単一のカスタム実行関数を管理するExpresserComponentです。 任意の処理ロジックをデリゲートとして設定し、条件に応じて実行できます。

DesiredExecutions

複数のカスタム実行関数を管理するExpresserComponentです。 複数の処理ロジックをリストで管理し、順次実行を行います。

ScreenOutSelfDestruction

オブジェクトがスクリーン外に出た時に自動的に破棄するExpresserComponentです。 スクリーン内に入ってからのみ作動するモードや、パディング設定など、 柔軟なスクリーン外判定機能を提供します。
