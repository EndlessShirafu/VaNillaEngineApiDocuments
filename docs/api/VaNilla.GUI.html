Table of Contents

Namespace VaNilla.GUI

Classes

Button

クリック可能なボタンコントロールを表します。 テキストの自動中央揃え機能を提供し、様々な用途に使用できる基本的なUIボタンです。

Control

すべてのGUIコントロールの基底クラスです。 位置、サイズ、外観、イベント処理などの基本機能を提供します。

ControlSkin

基本コントロール用のスキン定義を表すクラスです。 9パッチレンダリングシステムでコントロールの外観をカスタマイズします。

DXEx

VaNilla GUI用のDxLib拡張ユーティリティクラスです。 文字列の折り返し処理と描画の補助機能を提供します。

EventArguments

GUIイベントの基本イベント引数クラスです。 現在は空の実装ですが、将来的にイベント情報を追加できるようにクラスとして定義されています。

Form

ウィンドウフォームを表すコントロールです。 タイトルバーと各種ウィンドウボタンを持ち、ドラッグで移動可能なウィンドウを提供します。

FormSkin

Form用のスキン定義を表すクラスです。 ControlSkinを継承し、タイトルバーとウィンドウボタンのテクスチャを追加で定義します。

GraphicalUI

VaNilla GUIシステムの中核クラスです。 GUIメッセージの管理、フォーム管理、入力処理を統括します。

HScrollBar

水平スクロールバーコントロールを表します。 ScrollBarクラスを継承し、水平方向のスクロール機能を提供します。

Label

テキストを表示するためのラベルコントロールです。 自動折り返し機能を持ち、複数行のテキスト表示に対応しています。

ListControl<Type>

アイテムのリストを表示し、選択できるジェネリックリストコントロールです。 サムネイルとキャプション付きのアイテム表示、スクロール、単一/複数選択をサポートします。

ListControl<Type>.ListItem

リストコントロール内の各アイテムを表すクラスです。

MessageBox

メッセージボックスを表示するクラスです。 簡易的なダイアログを表示し、ユーザーの選択を取得できます。

MouseArgs

マウスイベントの引数を表すクラスです。 マウスの位置、移動量、クリック状態などの情報を保持します。

ResourceManager

GUIコントロールのリソース管理を行うクラスです。 XMLファイルからコントロールのデフォルト設定を読み込み、デフォルトインスタンスを提供します。

ScrollBar

スクロールバーコントロールの基底クラスです。 水平（HScrollBar）および垂直（VScrollBar）スクロールバーの共通機能を提供します。

ScrollBarSkin

ScrollBar用のスキン定義を表すクラスです。 ControlSkinを継承し、スクロールボタンのテクスチャを追加で定義します。

TextBox

テキスト入力を受け付けるテキストボックスコントロールです。 フォーカス取得時に自動的に入力受付を開始し、様々な入力モードをサポートします。

VScrollBar

垂直スクロールバーコントロールを表します。 ScrollBarクラスを継承し、垂直方向のスクロール機能を提供します。

XmlGUICreator

XMLファイルからGUIレイアウトを生成するクラスです。 Formを継承し、XML定義に基づいてコントロールを動的に配置します。

Structs

GUIMessage

GUIシステムで使用されるメッセージ構造体です。 マウス、キーボード、その他の入力イベント情報を保持し、コントロール間で伝達されます。

Enums

DialogResult

ダイアログの結果を表す列挙型です。

MessageBoxButtons

メッセージボックスに表示するボタンの種類を表す列挙型です。

MessageBoxIcon

メッセージボックスに表示するアイコンの種類を表す列挙型です。

TextAlign

テキストの配置方法を表す列挙型です。

Delegates

EventHandler

VaNilla GUIで使用されるイベントハンドラのデリゲートです。 注意：.NET標準のEventHandlerとは異なるカスタム定義です。
