ゲームパッド入力情報を管理するクラス。 スティック入力、ボタン状態、方向入力を追跡する。

InputInfoStruct

統合入力情報を管理する構造体。 マウス、キーボード、ゲームパッドの入力を統合的に管理し、 key_configの設定に基づいて抽象化された入力を提供する。

InputInfoValue

KeyboardInputInfo

キーボード入力情報を管理するクラス。 各キーの押下状態と継続時間を追跡する。

MouseInputInfo

マウス入力情報を管理するクラス。 マウスボタンの押下状態とカーソル位置を追跡する。

PlayerAbstructInput

プレイヤーの抽象的な入力を表すクラスです。

StaticInputInfo

静的入力情報アクセサクラス。 グローバルな入力状態にアクセスするための静的メソッドを提供する。 PlayerAbstractInputや入力記録システムで使用される。
