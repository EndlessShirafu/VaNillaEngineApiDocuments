コリジョンを持てる事を表象する。 Setupが呼ばれるより前に初期化パラメータを設定しておくとSetup時にコリジョンが生成される。 Setup後にコリジョン生成関数を呼び出すことで生成も可能。 コリジョンを「持っている」ではなくコリジョンを「持てる」事の表象になっているのは、 コリジョンを上記のどちらかの手順で生成していない場合、コリジョンを持っていない状態になるため。
