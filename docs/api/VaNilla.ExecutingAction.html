Classes

ActionCallbackAttribute

アクションコールバックの実行条件を指定するための属性です。 メソッドに付与することで、特定のフレーム範囲や間隔でコールバックを自動実行できます。

ActionCallbackAttributeCollector

アクションコールバック属性とステータス変更コールバック属性を収集・管理するクラスです。 リフレクションを使用してアセンブリ内の属性情報を収集し、高速なアクセスを提供します。

HasActionCallback

アクションコールバックを持つことの表明を行う

StatusChangedCallbackAttribute

ステータス変更時のコールバックを指定するための属性です。 メソッドに付与することで、オブジェクトのステータスが変更された際に自動的にコールバックが実行されます。
