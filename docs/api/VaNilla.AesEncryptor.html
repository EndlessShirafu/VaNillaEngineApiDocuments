Remarks

このクラスは、AES（Advanced Encryption Standard）アルゴリズムを使用して、 バイト配列や文字列の暗号化・復号化機能を提供します。

主な特徴:

  • 128ビットのブロックサイズ（AES標準）
  • 自動的なIV（初期化ベクトル）生成
  • IVを暗号化データの先頭に含める方式
  • UTF-8エンコーディング対応

セキュリティに関する注意:

  • このクラスはデフォルトの固定キーを使用しています。本番環境では適切なキー管理が必要です。
  • 暗号化キーは環境変数、設定ファイル、または安全なキー管理システムから取得することを推奨します。

Parameters

Returns

暗号化されBase64エンコードされた文字列

Remarks

このメソッドは以下の処理を行います：

  1. 文字列をUTF-8でバイト配列に変換
  2. バイト配列を暗号化
  3. 暗号化されたバイト配列をBase64文字列に変換

Base64形式を使用することで、暗号化データを文字列として安全に保存・転送できます。
