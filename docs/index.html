🎮 VaNillaエンジンとは

VaNillaエンジンは、既存のゲームエンジン（DxLib・Unity）をバックエンドとして動作する 「メタ・ゲームエンジン」 です。
最新のC#＆.NETの機能をフル活用した、高性能・高機能な2Dゲーム開発特化のAPIを提供します。
さらに、同じコードがDxLibとUnityの両方で動作し、プラットフォームに依存しない2Dゲーム開発を実現します。（※Unity版は目下開発中）

✨ なぜVaNillaを使うのか？

  • 🧩 宣言的プログラミング - 属性＆YAMLでコンポーネントを宣言するだけで必要機能を自動構築
  • ⚡ 超高速オブジェクト処理 - 1,000個以上のオブジェクトでも60FPS維持
  • 🔄 プラットフォーム非依存 - DxLibで開発したゲームをUnityに移植（またはその逆）が簡単（※Unity版は開発中）
  • 🎯 2Dゲーム特化 - シューティング、アクション、パズルなど2Dゲームに必要な機能を完備
  • 🏆 実戦実績あり - Steam発売タイトル「∀stralbringer」で採用

🎯 ExpresserComponent-Oriented Programming

VaNillaの最大の特徴は、「表現」を中心としたコンポーネント設計です。

従来のEntity-Component-System vs ExpresserComponent

// ❌ 従来の設計：「何を持っているか」で構成
class Enemy : MonoBehaviour {
    Transform transform;      // 位置データ
    SpriteRenderer renderer;  // 描画データ
    Collider2D collider;     // 衝突データ
    int hp;                  // HPデータ
}

// ✅ VaNillaの設計：「何を表現するか」で構成
[DefaultExpresserComponent(typeof(HasPosition))]     // 「位置を持つ」という表現
[DefaultExpresserComponent(typeof(HasTexture))]      // 「見た目を持つ」という表現  
[DefaultExpresserComponent(typeof(HasCollision))]    // 「ぶつかる」という表現
[DefaultExpresserComponent(typeof(HasHitPoint))]     // 「HPを持つ」という表現
public partial class Enemy : Object { }

なぜこの設計が優れているのか？

  1. 依存関係の自動解決 - HasCollisionは自動的にHasPositionを要求
  2. 機能の着脱が簡単 - 属性を追加/削除するだけ
  3. コードの再利用性 - ExpresserComponentは他のObjectでも使える
  4. 実行順序の最適化 - システムが最適な順序で処理

🎮 実装例集

例1: プレイヤーキャラクター（キーボード操作）

[AutoGenerateConstructor]
[DefaultExpresserComponent(typeof(HasPosition), auto_member_generate: true)]
[DefaultExpresserComponent(typeof(HasTexture), auto_member_generate: true)]
[DefaultExpresserComponent(typeof(HasCollision), auto_member_generate: true)]
[DefaultExpresserComponent(typeof(KeyboardControl), auto_member_generate: true)]
public partial class Player : Object
{
    private float speed = 5.0f;
    
    public override bool SetUp()
    {
        // 初期設定
        hasPosition.Position = new Point2(320, 400);
        hasTexture.Texture = TextureResourceType.Player;
        hasCollision.Shape = new CircleCollisionShape(16);
        
        // キーボード操作設定
        keyboardControl.SetMoveKeys(
            KeyCode.Left, KeyCode.Right,
            KeyCode.Up, KeyCode.Down
        );
        keyboardControl.MoveSpeed = speed;
        
        return true;
    }
    
    // 衝突時の処理
    public override void OnCollision(Object other)
    {
        if (other is Enemy)
        {
            // ダメージ処理
            TakeDamage(10);
        }
    }
}

例2: 弾幕パターン（ObjectFactory使用）

[ExecuteOnlyInActive]  // アクティブ時のみ実行
public partial class BulletSpawner : Object
{
    private int frameCount = 0;
    
    public override void Execute()
    {
        frameCount++;
        
        // 30フレームごとに円形弾幕を発射
        if (frameCount % 30 == 0)
        {
            SpawnCircularBullets(16, 200);
        }
    }
    
    private void SpawnCircularBullets(int count, float radius)
    {
        for (int i = 0; i < count; i++)
        {
            float angle = (float)(2 * Math.PI * i / count);
            
            // ObjectFactoryで高速生成
            var bullet = ObjectFactory.Activate<Bullet>();
            bullet.Initialize(
                position: hasPosition.Position,
                velocity: new Vector2(
                    (float)Math.Cos(angle) * 3,
                    (float)Math.Sin(angle) * 3
                )
            );
        }
    }
}

例3: GUI設定画面

public partial class SettingsScreen : Object
{
    private Form settingsForm;
    private Slider bgmSlider;
    private Slider seSlider;
    private Button applyButton;
    
    public override bool SetUp()
    {
        // 設定画面フォームを作成
        settingsForm = new Form("ゲーム設定", 400, 300);
        settingsForm.Position = new Point2(200, 150);
        
        // BGM音量スライダー
        bgmSlider = new Slider(0, 100, SoundManager.BgmVolume);
        bgmSlider.Position = new Point2(50, 50);
        bgmSlider.ValueChanged += OnBgmVolumeChanged;
        settingsForm.AddControl(bgmSlider);
        
        // SE音量スライダー
        seSlider = new Slider(0, 100, SoundManager.SeVolume);
        seSlider.Position = new Point2(50, 100);
        seSlider.ValueChanged += OnSeVolumeChanged;
        settingsForm.AddControl(seSlider);
        
        // 適用ボタン
        applyButton = new Button("適用", 100, 30);
        applyButton.Position = new Point2(150, 200);
        applyButton.Click += OnApplyClick;
        settingsForm.AddControl(applyButton);
        
        return true;
    }
    
    private void OnBgmVolumeChanged(object sender, EventArgs e)
    {
        SoundManager.SetBgmVolume(bgmSlider.Value);
    }
}
